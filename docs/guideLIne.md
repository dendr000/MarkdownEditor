# 프로젝트 시작 가이드

본 프로젝트를 로컬 환경에서 실행하기 위한 초기 설정 및 구동 방법입니다.

## 1. 사전 요구 사항 (Prerequisites)
프로젝트를 실행하기 전에 아래 환경이 구축되어 있어야 합니다.
- **Node.js**: v24.16.0 이상

## 2. 프로젝트 가져오기
작업 상황에 맞게 프로젝트 코드를 로컬 환경으로 가져옵니다.

### 최초 1회 실행 시 (Clone)
새로운 PC에서 프로젝트를 처음 내려받을 때 사용합니다.  
**```bash**    
git clone <GitHub 저장소 주소>
cd <프로젝트 폴더 이름>

### 이미 존재하는 경우
기존에 클론한 폴더에서 최신 코드로 동기화할 때 사용합니다.
```
git pull origin main
```

## 라이브러리 패키지 설치
프로젝트 구동에 필요한 의존성 패키지를 설치합니다.
```
npm install
```

## 로컬 개발 서버 구동
개발 서버를 실행하여 프로젝트를 확인합니다.
```
npm run dev
```
